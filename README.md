## Tech stack

- Backend: WCF Service (.NET Framework 4.5.1, C#)
- Frontend: ExtJS (MVC)
- Database: SQL Server
- Docs: MkDocs + Material for MkDocs

# Description

This project consists of two main parts:

1. `docs/`: MkDocs documentation for the **Quản lý sinh viên** project architecture.  
2. `quanlysinhvien/`: Application source code (WCF backend + ExtJS frontend).

---

## docs

Run the documentation locally with MkDocs:

```pwsh
cd docs
mkdocs serve
```

Then open the URL shown in the terminal (usually `http://127.0.0.1:8000/`) in your browser.

---

## quanlysinhvien

Open the solution in **Visual Studio** (not Visual Studio Code).

To start the application:

```text
Press F5 to build and run (Debug).
```
